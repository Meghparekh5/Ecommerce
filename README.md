# Ecommerce Fashion Website

A modern and responsive Ecommerce Fashion Website built using HTML, CSS, Bootstrap, and Bootstrap Icons. This project showcases a stylish fashion landing page with trending products, hero banners, customer reviews, blog sections, and a fully responsive layout.

---

## 🚀 Features

* Modern Ecommerce UI Design
* Fully Responsive Layout
* Bootstrap 5 Framework
* Hero Banner Section
* Trending Products Section
* Explore by Colors Section
* Customer Review/Testimonial Section
* Blog Showcase Section
* Clean Footer with Newsletter Form
* Bootstrap Icons Integration
* Custom Styling Support

---
